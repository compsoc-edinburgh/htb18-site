Code Golf

What is Code Golf?

The idea behind code golf is simple. Write a program to complete the given challenge in any language you like: the shortest correct code wins!

The Challenge

You've doubtless heard of base 10 (decimal), base 2 (binary), and base 16 (hexadecmial), but have you heard of base Fibonacci? All the cool software engineers are using it nowadays!

So you know how in base ten, the value of each digit from right to left is the nth power of ten (1, 10, 100, 1000, ...); in base two, the value is the nth power of two (1, 2, 4, 8, ...); well in base Fibonacci, the value is the nth Fibonacci number (1, 1, 2, 3, 5, 8, ...)! Like binary, numbers in base Fibonacci only use the digits 0 and 1.

This does however mean there's multiple ways of reperesenting the same number, for example 1101FIB = (3 + 2 + 0 + 1)DEC = 6DEC, and 10001FIB = (5 + 0 + 0 + 0 + 1)DEC = 6DEC. This has broken my old equals function, so can you write me a new one please?

Input (from stdin): The Fibonacci representation of n and m, separated by a comma
Output (to stdout): 'T' if n == m, 'F' otherwise (not including quotes)
Constraints: 0 < n < 1000DEC, 0 < m < 1000DEC
